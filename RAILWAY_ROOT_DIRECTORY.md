# ⚠️ ВАЖНО: Настройка Root Directory в Railway

## Проблема
Railway пытается собрать весь репозиторий (включая client), потому что это npm workspace. Это вызывает ошибки с Next.js.

## Решение

### В Railway UI:

1. Открой сервис `adapted-russia-server`
2. Перейди в **Settings** → **Service**
3. Найди **Root Directory**
4. **ОБЯЗАТЕЛЬНО установи**: `server`
5. Сохрани изменения

### Проверка:

После установки Root Directory:
- Railway будет собирать только папку `server/`
- Не будет видеть `client/` и его зависимости
- Сборка должна пройти успешно

### Если Root Directory не установлен:

Railway будет:
- ❌ Видеть весь репозиторий
- ❌ Пытаться собрать workspace
- ❌ Видеть Next.js из client
- ❌ Получать ошибки с package-lock.json

### После настройки:

Railway автоматически перезапустит деплой. Проверь логи - должна быть только сборка server.
